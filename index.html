Variables

Plus Calculator

Esta calculadora ofrece características adicionales en comparación con las calculadoras básicas. A continuación, se presenta un breve manual de uso para aprovechar estas funciones adicionales.

Características

  • Introducción de valores mediante botones y teclado.
  • Multiplicación implícita con el uso de paréntesis.
  • Almacenado y uso de variables mediante el "Session Storage".

Uso de Variables

Para inicializar unas de las 3 variables disponibles, se debe seguir la siguiente estructura:

[variable]SET[valor]

Por ejemplo:

XSET10Asigna el valor "10" a la variable "X"

Atajos de teclado

Todos los botones tienen asignado el mismo valor que en el teclado, en caso de los botones especiales, se asignaron nuevas teclas o combinaciones:

  • DEL = ← (Backspace)
  • AC = Supr (Suprimir)
  • ANS = Alt + A
  • SET = Alt + S
